Follow these simple steps to set up VPN on your iPhone or iPad:

Step 1: Install the App

Open the App Store and install Streisand (recommended) or V2Box.

Step 2: Copy the VLESS Link

Go back to the bot and copy the VLESS link you received. It looks like this:

vless://xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx@...

Step 3: Import Configuration

Open the Streisand app and tap the "+" button in the top right corner, then select "Import from clipboard".

Step 4: Connect

Select the added server from the list and tap the connect button. On first connection, iOS will ask permission to add VPN configuration - tap "Allow".

Step 5: Done!

A VPN icon will appear in the status bar. Now you can freely browse the internet!

Using on Other Devices

The process is similar on other platforms: install the app, import the VLESS link, and connect.

Recommended Apps:

  • Android: v2rayNG (Google Play or GitHub)
  • Windows: Nekoray
  • macOS: Streisand (App Store) or Nekoray
  • Linux: Nekoray or v2rayN
