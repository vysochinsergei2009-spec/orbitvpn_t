iOS / iPadOS Setup

Step 1: Install App

Open the App Store and install V2RayTun (recommended) or Streisand

Step 2: Copy VLESS Link

Go back to the bot and copy the VLESS link you received. It looks like this:

vless://xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx@...

Step 3: Import Configuration

Open V2RayTun app ‚Üí tap "+" button (top right) ‚Üí select "Import from clipboard"

Step 4: Connect

Select the added server from the list and tap Connect. On first connection, iOS will ask permission to add VPN configuration ‚Äî tap "Allow"

Step 5: Done!

A VPN icon will appear in the status bar. Now you can freely browse the internet!

  • After connecting, verify it works by opening any website
